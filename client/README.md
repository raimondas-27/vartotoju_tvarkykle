# Simple users list api implementation

Simple users api table

## Features

1.view all users
2.add user to list 
3.remove user from list
4.edit good user in list



### Installation

1. clone repository 
2. npm install 
3. user localhost

### Dev scripts

'npm start' start front-end application for local development.

### Dependencies

-react-router-dom
-font-awesome@4.7.0 
-react-toastify 
-axios 
-bootstrap 4.6.0
